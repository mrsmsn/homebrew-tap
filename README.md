# homebrew-tap

Homebrew tap for [mrsmsn](https://github.com/mrsmsn)'s tools.

## issue

A local-first issue-management CLI (`issue`) with a lazygit-style TUI (`lazyissue`).

```sh
brew install mrsmsn/tap/issue
```

Installs both the `issue` and `lazyissue` commands, plus shell completions
(bash/zsh/fish) for `issue` — so `issue <Tab>` works with no extra setup.

Source: https://github.com/mrsmsn/issue — the formula here is published
automatically on each release tag.
